# STM32F103C8T6-SnakeGame

# 基于STM32的体感贪吃蛇游戏

## 项目概述

本项目基于 **STM32F103C8T6** 微控制器，利用 **MPU6050** 六轴姿态传感器和 **SSD1306** OLED显示屏，实现了一款通过体感控制的贪吃蛇游戏。玩家通过倾斜开发板来控制蛇的移动方向，游戏分数和最高分可保存至片上Flash，掉电不丢失。

项目融合了**卡尔曼滤波**和**马哈尼滤波**两种姿态解算算法，以获取稳定、准确的欧拉角，从而实现对游戏方向的精准控制。

---

## 硬件连接

| 外设         | 接口        | STM32引脚        |
| ------------ | ----------- | ---------------- |
| SSD1306 OLED | I2C1        | PB6 (SCL), PB7 (SDA) |
| MPU6050      | I2C2        | PB10 (SCL), PB11 (SDA) |
| 用户按键（扩展）| I2C1 扩展IO | 通过I2C1读取0x42设备 |
| 调试串口     | USART1      | PA9 (TX), PA10 (RX) |

> 注：OLED和MPU6050使用不同的I2C总线，避免地址冲突。按键通过I2C总线扩展芯片读取。

---

## 功能说明

1. **游戏主界面**：开机显示“START”选项和游戏Logo，通过MPU6050倾斜选择开始。
2. **体感控制**：  
   - 倾斜板子控制蛇头方向（上/下/左/右）。  
   - 使用卡尔曼滤波和马哈尼滤波融合加速度计和陀螺仪数据，获得平滑姿态角。
3. **分数记录**：  
   - 实时显示当前分数和最高分。  
   - 最高分保存在Flash地址 `0x0801f000`，支持掉电存储。
4. **游戏逻辑**：  
   - 蛇身长度每吃一个食物增加1。  
   - 撞墙或撞自身游戏结束，显示“GAME_OVER”。  
   - 按中间按键（Key_MID）返回主菜单重新开始。
5. **串口调试**：通过USART1打印姿态角数据（波特率9600）。

## 主要模块介绍

### 1. 姿态解算模块
- **kalman.c / kalman.h**：卡尔曼滤波器实现，用于融合加速度计和陀螺仪数据，计算roll/pitch角。
- **mahony_filter.c / mahony_filter.h**：马哈尼滤波器实现，提供互补滤波姿态解算，输出pitch、roll、yaw。
- 在 `my_snack.c` 中同时调用两种滤波器，可根据需要选择使用。

### 2. 显示模块
- **SSD1306.c / SSD1306.h**：OLED驱动，支持画点、画线、显示字符、字符串、中文字库、BMP图片等。
- **font.c / font.h**：包含ASCII字库（6×8和8×16）以及自定义中文字库（16×16），用于显示菜单和提示信息。

### 3. 贪吃蛇游戏逻辑
- **my_snack.c / my_snack.h**：游戏核心逻辑。  
  - 蛇身坐标数组管理。  
  - 食物随机生成（避开边界）。  
  - 方向控制通过 `Dirtion()` 函数根据MPU6050姿态角更新。  
  - 分数更新及最高分Flash读写。

### 4. Flash存储
- **Flash.c / Flash.h**：封装STM32内部Flash擦除和半字写入/读取函数。  
  - 起始地址 `FLASH_START_ADDR = 0x0801f000`（需确保不与其他代码冲突）。

### 5. 通信模块
- **Serial.c / Serial.h**：USART1驱动，支持 `printf` 重定向和自定义数据包收发。
- **my_i2c.c / my_i2c.h**（未提供代码但被引用）：I2C底层读写，用于OLED、MPU6050和按键扩展芯片。

### 6. 延时与定时器
- **Delay.c / Delay.h**：基于SysTick的微秒/毫秒/秒级延时。
- **Time.c / Time.h**：TIM2定时器中断，用于游戏主循环的周期性方向检测（约50ms一次）。

## 使用说明

### 环境搭建
- 开发板：STM32F103C8T6最小系统板。
- 工具链：Keil MDK 5（或STM32CubeIDE），标准外设库（STM32F10x_StdPeriph_Driver）。
- 连接OLED和MPU6050至对应I2C引脚。

### 编译与下载
1. 将所有 `.c` 和 `.h` 文件添加到工程。
2. 确保包含路径正确（`include.h` 汇总了所有头文件）。
3. 配置工程使用ST-Link或串口下载。
4. 编译并烧录。

### 操作指南
- **上电**：OLED显示“START”箭头和Logo。  
- **选择开始**：倾斜板子使箭头指向“START”区域，按下中间按键（Key_MID）进入游戏。
- **游戏控制**：  
  - 向前倾斜（俯仰角>10°）→ 向上  
  - 向后倾斜（俯仰角<-10°）→ 向下  
  - 向左倾斜（横滚角>10°）→ 向左  
  - 向右倾斜（横滚角<-10°）→ 向右  
- **游戏结束**：按中间键返回主菜单。

### 参数调整
- 姿态角灵敏度阈值可在 `Dirtion()` 函数中修改（当前为±10度）。
- 游戏速度（蛇移动间隔）由主循环中的 `Num % 50 == 0` 控制，可通过调整TIM2中断周期或取模值改变。

---

## 注意事项

1. **I2C地址**：OLED地址为 `0x7A`，MPU6050地址为 `0xD0`，按键芯片地址为 `0x42`（需根据实际硬件调整）。
2. **Flash写入**：每次擦除一页（1KB），频繁写入可能影响寿命，仅最高分更新时写入。
3. **姿态解算**：卡尔曼和马哈尼滤波参数已调至较稳定状态，如需修改可调整 `kalman.c` 中的 `Q_angle`、`R_measure` 等参数，或 `mahony_filter.c` 中的 `Kp`、`Ki`。
4. **内存占用**：贪吃蛇最大长度255，使用 `snake_x[255]` 和 `snake_y[255]` 数组，注意堆栈空间。
